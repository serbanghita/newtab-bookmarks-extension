# New Tab Unlimited Bookmarks

> Add unlimited bookmarks to your browser's new tab page. Works with your existing bookmark folders. \
> Available for Google Chrome and Microsoft Edge.

## Run from source

1. `git clone git@github.com:serbanghita/newtab-bookmarks-extension.git`
2. Open your browser's extensions page — `chrome://extensions/` in Chrome or `edge://extensions/` in Edge — and enable **Developer mode**.
3. Click **Load unpacked** and select the cloned `newtab-bookmarks-extension` folder.

## Contributing

Pull requests are welcome. The recommended flow uses a sandboxed Chrome for Testing instance so you don't touch your real profile or bookmarks.

1. Fork the repo and create a branch off `main`.
2. `npm install`
3. `npm run preview` — builds the extension, downloads Chrome for Testing into `.chrome-for-testing/`, seeds a throwaway profile in `.chrome-dev-profile/` with sample bookmarks, and launches Chrome with the extension loaded. esbuild stays in watch mode, so edits in `src/` rebuild automatically; reload the new tab to see changes.
4. `npm run lint` before committing.
5. Open a PR against `main` describing the change. UI changes: please include a screenshot (and a dark-mode screenshot if you're touching a theme).

Notes:

- `npm run dev` runs the esbuild watcher only, if you'd rather load the extension into your own browser via `chrome://extensions/` or `edge://extensions/` → "Load unpacked".
- Only `dist/js/newtab.js` is generated by the build. CSS and HTML under `dist/newtab/` and `dist/popup/` are hand-edited and committed.
- No test suite — verify changes by exercising them in the preview window.
